固井水泥石强度测定方法

发表于 2018-07-24 分类于 固井

模具的准备

强度试件模具的准备应遵守以下原则：上下底面与玻璃板的接触表面及模具的内表面应均匀地抹上一层薄薄的黄油，以便养护时不漏水。应特别注意黄油不宜涂抹过厚，否则不能使水泥填满模具，出现坑凹等不规整现象会影响水泥石强度的测定。

水泥浆制备

制备好的水泥浆应立即注入常压稠度仪，在90℃条件下稠化20分钟，目的有二：一是稠化升温过程可以使水泥浆中的气泡消除，以便形成致密的水泥石。二是可以通过这个过程观察浆体是否稳定。

灌注模具

将稠化好的水泥浆倒入准备好的模具中至模具高度的二分之一，并用细的玻璃棒搅拌，使水泥浆能填满模具的边缘及模具与玻璃板形成的夹缝。应先把水泥浆注入所有模具中，再进行搅拌操作，接着再把水泥浆注满模具，同样进行搅拌操作。最后将模具顶部盖上涂有黄油的玻璃板，每一个配方的水泥石试件应制备两个及以上，以减少偶然因素带来的误差。

试件的养护

将制备好的模具放入养护箱中养护，并根据实验方案设置温度，例如实验设置的温度为90℃，养护时间为48小时。一切准备就绪后，开始计算养护时间。

试件的修善

当水泥石试件从养护箱中取出后，脱去模具，轻轻刮去水泥石试件外的黄油、用砂纸磨平样品的上下底面，确保试件上无倾斜或凸凹不平等形貌。

水泥石强度测定

使用抗压强度实验机对试件进行抗压试验。对普通强度的试件，加载率为每分钟400磅/英寸2（16000磅力）（71.7千牛），对低强度试件，其抗压强度等于或低于500磅/英寸2（3.5MPa）的情况，加载率为每分钟1000磅/英寸2（4000磅力）（17.9千牛），在接近极限强度时，不要再调整机器控制装置。每个配方的读数取该样品两个试件读数的平均值作为该样品的测量值。

DFC-0710B型增压稠化仪操作流程

发表于 2018-07-21 更新于 2019-05-14 分类于 固井

开始阶段

1、关闭稠化仪所有开关，接通电源开关。

2、编辑压力控制程序，载入新的程序。按换页键调至程序编辑页，按转换键选择菜单中的参数。使用增值键或减值键改变参数值。

3、制浆，组合好浆杯。

4、使用专用夹子，把浆杯放入釜内，然后接通电机开关，浆杯应转动平稳。（装浆杯时，注意不要磕碰到金属O型圈）。

5、使用专用夹子钩住电位计，装入釜内，调制电压表初始稠度值稳定。

6、旋上釜盖，用橡皮锤轻磕紧。旋入热电偶，留有适当气隙。

7、接通增压泵开关，接通气至油箱开关。直到内热电偶螺丝处溢出油，釜内油满。旋紧螺丝。

8、按动AUTO/MAN自动/手动转换键(显示MAN)，按一次RUN/HOLD运行/暂停键（RUN显示），温控器控制程序开始运行。再按动AUTO/MAN自动/手动转换键。

9、打开加热器，按自动键，实验正式开始。

结束阶段

1、实验结束时，关闭自动开关，复位控制程序（按RUN/HOLD运行/暂停键持续2秒以上，RUN消失）。

2、开启冷却水开关，开始冷却至浆杯热电偶温度低于95℃。

3、缓慢开启高压释放开关阀，接通空气至釜开关，釜内油排净时，会发出排气声。这时关闭空气至釜开关。

4、排气声消失后，方可旋出热电偶。借助胶锤，旋出釜盖，钩出电位计和浆杯。

5、将杯内的水泥倒掉，清洗浆杯、浆叶、隔膜片 等，晾干重新涂油酯备用。

OWC-9040F型增压稠化仪操作流程

发表于 2018-07-21 更新于 2018-07-24 分类于 固井

开始阶段

1、关闭稠化仪所有开关，接通电源开关。

2、编辑压力控制程序，载入新的程序。按换页键调至程序编辑页，按转换键选择菜单中的参数。使用增值键或减值键改变参数值。

3、制浆，组合好浆杯。

4、使用专用夹子，把浆杯放入釜内，然后接通电机开关，浆杯应转动平稳。（装浆杯时，注意不要磕碰到金属O型圈）。

5、使用专用夹子钩住电位计，装入釜内，调制电压表初始稠度值稳定。

6、旋上釜盖，用橡皮锤轻磕紧。旋入热电偶，留有适当气隙。

7、打开增压泵，接通气至油箱开关。用毛巾覆盖热电偶直到内热电偶螺丝处溢出油，釜内油满。旋紧螺丝。

8、打开加热器。

9、调节稠化仪上的仪表，开始运行。

10、打开加热器，按自动键，实验正式开始。

结束阶段

1、稠度升至30BC，关闭加热器和电机，复位控制程序（按RUN/HOLD运行/暂停键持续2秒以上，RUN消失）。

2、开启冷却水开关，开始冷却至浆杯热电偶温度低于95℃。

3、关闭空气源阀和压缩机。

4、打开空气排放阀，待小压力表压力降至0时，打开高压释放阀，大压力表降至0时，打开空气至釜回油，当听到空气排放的嗡嗡声时，关闭空气至釜。

4、排气声消失后，方可旋出热电偶，注意用毛巾覆盖，以免油喷出。借助胶锤，旋出釜盖，钩出电位计和浆杯。

5、将杯内的水泥倒掉，清洗浆杯、浆叶、隔膜片等，晾干重新涂油酯备用。
